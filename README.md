# ♡ Carta para ella

Una carta interactiva hecha con amor, inspirada en el universo de Taylor Swift.

## ¿Qué es esto?

Una experiencia de lectura página por página, diseñada para que quien la recibe la descubra de a poco — sin poder saltear, sin distracciones. Cada sección tiene su propia estética inspirada en una era de Taylor Swift.

| 7 | ♡ Para siempre |

## Cómo funciona

- Fondo estrellado animado
- Cada página aparece con animación suave al hacer clic
- Barra de progreso discreta
- Tipografía manuscrita (Caveat + Kalam)
- Diseño adaptado a celular

## Cómo usar

No necesita instalación ni dependencias. Solo abrí el archivo `index.html` en cualquier navegador, o accedé al link de GitHub Pages.

## Tecnologías

- HTML5 / CSS3 / JavaScript vanilla
- Google Fonts (Caveat, Kalam)
- Canvas API para las estrellas
